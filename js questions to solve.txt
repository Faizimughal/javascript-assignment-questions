ğŸ”¤ Basic Info Methods

.length â†’ property, not a method, gives number of characters.

.charAt(index) â†’ character at given position.

.charCodeAt(index) â†’ UTF-16 code of character.

.codePointAt(index) â†’ Unicode code point of character.

String.fromCharCode(num1, num2, â€¦) â†’ create string from UTF-16 codes.

String.fromCodePoint(num1, num2, â€¦) â†’ create string from Unicode points.

ğŸ” Searching & Checking

.indexOf(substring, start?) â†’ first occurrence index.

.lastIndexOf(substring, start?) â†’ last occurrence index.

.includes(substring, start?) â†’ true/false if found.

.startsWith(substring, start?) â†’ check beginning.

.endsWith(substring, length?) â†’ check ending.

.match(regex) â†’ array of matches.

.matchAll(regex) â†’ iterator of matches.

.search(regex) â†’ index of first regex match.

âœ‚ï¸ Extracting Parts

.slice(start, end?) â†’ substring by index.

.substring(start, end?) â†’ similar to slice (no negatives).

.substr(start, length?) â†’ old, not recommended, but still works.

ğŸ›  Manipulating

.replace(searchValue, newValue) â†’ replace first match.

.replaceAll(searchValue, newValue) â†’ replace all matches.

.concat(str1, str2, â€¦) â†’ join strings (but + is used more).

.repeat(count) â†’ repeat string.

.padStart(targetLength, padString?) â†’ add padding at start.

.padEnd(targetLength, padString?) â†’ add padding at end.

.trim() â†’ remove spaces from both ends.

.trimStart() / .trimLeft() â†’ remove leading spaces.

.trimEnd() / .trimRight() â†’ remove trailing spaces.

.split(separator, limit?) â†’ convert into array.

ğŸ”„ Changing Case

.toUpperCase() â†’ all uppercase.

.toLowerCase() â†’ all lowercase.

.toLocaleUpperCase() â†’ locale-based uppercase.

.toLocaleLowerCase() â†’ locale-based lowercase.

ğŸ“¦ Other Utilities

.valueOf() â†’ primitive string value.

.toString() â†’ convert to string (works on objects too).

.normalize(form?) â†’ Unicode normalization.

.localeCompare(str) â†’ compare strings according to locale.

.at(index) â†’ newer method, allows negative indexing.
